# Hussein Portfolio (Static Site)

This is a simple one‑page portfolio (HTML/CSS/JS).  
Files:
- index.html
- styles.css
- script.js
- assets/ (profile photo, CV, favicon)

## Run locally
Just open `index.html` in your browser.

## Deploy on Netlify (recommended)
1. Go to Netlify.
2. Drag & drop the **hussein-portfolio** folder (or the zip contents) into Netlify Drop.
3. Netlify will publish it and give you a live URL.

## Deploy on GitHub Pages
1. Create a new repo (e.g. `portfolio`).
2. Upload all files (index.html, styles.css, script.js, assets/).
3. In GitHub repo settings → Pages:
   - Source: Deploy from a branch
   - Branch: `main` / root
4. Your site will be available at: https://<username>.github.io/<repo>/
